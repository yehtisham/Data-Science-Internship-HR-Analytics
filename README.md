# SIEHS HR Analytics Dashboard

This project analyzes workforce attendance and punctuality patterns for Sindh Integrated Emergency & Health Services (SIEHS).  
The analysis uses HR data for approximately 1,800 employees across multiple departments, focusing on punctuality trends, shift compliance, and overall workforce efficiency.

### Objectives
- Identify attendance and lateness trends using HR timestamp data.
- Develop key metrics such as on-time arrival rate, average delay, and department-level punctuality.
- Visualize performance improvements over time.

### Tools & Technologies
- **Power BI** for dashboard design and KPI visualization.
- **Python / Excel** for initial data cleaning and feature preparation.
- **LaTeX** for professional documentation and reporting.

### Key Outcomes
- Built an interactive HR analytics dashboard.
- Improved punctuality monitoring, contributing to a **10% increase in on-time attendance**.
- Enabled data-driven management decisions through visual insights.

### Repository Contents
- `Dashboard.pbix`: Power BI dashboard file.
- `report/report.pdf`: Final report written in LaTeX.
